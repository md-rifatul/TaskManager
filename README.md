# 📝 Task Manager API

A simple ASP.NET Core Web API for managing tasks. This project demonstrates basic CRUD operations using an in-memory list and is ideal for learning and prototyping purposes.

---

## 🚀 Features

- ✅ Get all tasks
- 🔍 Get task by ID
- ➕ Add a new task
- 📝 Update a task
- ❌ Delete a task
